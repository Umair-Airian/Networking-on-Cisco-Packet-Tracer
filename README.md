# Cisco Packet Tracer Labs

This repository contains my networking practice labs using Cisco Packet Tracer.

## Labs Included
- IP and Subnetting
- VLAN Configuration
- Static and Dynamic Routing
Explaination:
These files include the real world scenarios like how switches works
actually and how we sent packet to the routers and how we assign Ip

## How to Use
- Open the `.pkt` files in Cisco Packet Tracer.
- Read the lab objectives below before opening.
